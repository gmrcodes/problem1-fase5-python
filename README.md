Fase 5 - Problema 1: Sistema de Evaluación de Nivel de Compromiso de Sesión

Bienvenido al repositorio del Problema 1 - Fase 5 del curso Fundamentos de Programación 213022.

Este proyecto demuestra la aplicación de conceptos fundamentales de Python para la resolución de problemas por computadora.
También se muestra la comprensión de la sintaxis y semántica de las estructuras de control utilizando un lenguaje de programación bajo el paradigma estructurado, en este caso Python.

📋 Descripción del Proyecto

El objetivo principal de este ejercicio es automatizar la evaluación del nivel de compromiso de los usuarios basándose en métricas de comportamiento (duración de sesión y cantidad de clics). El software procesa una estructura de datos bidimensional (matriz) y aplica reglas de negocio mediante funciones para clasificar a cada cliente en niveles: Alto, Medio o Bajo.

Conceptos Aplicados:
  Estructuras de Datos: Manejo de matrices mediante listas anidadas.
  Lógica Condicional: Uso de operadores lógicos (and, or) para la toma de decisiones.
  Modularización: Creación de funciones con paso de parámetros y retorno de valores.
  Documentación: Implementación de Docstrings y buenas prácticas.
  Aseguramiento de Calidad: Pruebas de escritorio manuales y validación de tipos de datos.
